Multiplayer Car Basic

Multiplayer Car Basic is a simple local multiplayer driving game developed using Unity as part of the Create With Code course.

This project focuses on learning core gameplay programming concepts, including player movement, camera systems, and split-screen multiplayer functionality.

What I Learned
- Player movement and input handling
- Camera follow behavior
- Switching between camera views
- Local multiplayer implementation
- Scene setup and game logic basics

Controls

Player 1

- Move: WASD
- Switch Camera: F

Player 2

-Move: Arrow Keys
- Switch Camera: K
